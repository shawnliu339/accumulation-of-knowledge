# iterm2常用快捷键
1. 移动到行首
```
Ctrl + a
```

2. 移动到行尾
```
Ctrl + e
```

3. 向前移动个单词
```
esc + b
```
通过iterm key mapping后(参考[iterm2-zsh.md#Install#6](iterm2-zsh.md))
```
option + 方向键
```

1. 向后移动个单词
```
esc + f
```
通过iterm key mapping后
```
option + 方向键
```

1. 删除到行首
```
Ctrl + u
```

1. 删除一个单词
```
Ctrl + w
```

1. 撤销
```
Ctrl + -
```
