# Intellij Short Cut
历史记录复制: cms shift v
for, if surround with: cmd opt t
继承层级显示: ctr + H
换类型: cmd + F6
打开关闭折叠: cmd shift + / -
移动到下一个代码块(大括号迅速匹配): cmd opt [ / ]
